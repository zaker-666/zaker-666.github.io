如何用hugo搭建个人博客并上传到GITHUB

January 6, 2020

1.从GITHUB上下载hugo

下载地址[]( https://github.com/gohugoio/hugo/releases/tag/v0.62.2 )

2.解压安装并安装

解压安装包到任意盘，将安装包所安装的文件夹添加至windows环境变量中

3.为hugo添加主题

在安装目录下打开命令行，使用如下命令为hugo 快速安装主题

git init
git submodule add https://github.com/budparr/gohugo-theme-ananke.git themes/ananke

将主题添加到配置中

在安装文件夹中找到config.toml文件，使用记事本打开，在后面添加theme = “ananke” 并保存。

3.新建一篇博客

在命令行运行

hugo new posts/my-first-post.md

你就可以在安装文件夹中的content/posts文件夹中找到名为my-first-post.md的文件，打开并编辑，编辑完成后修改标题中dratf:ture为draft:false

4.将博客上传到GITHUB

1.在自己的github中新建一个名为：用户名.github.io的仓库

2.回到hugo安装目录运行如下命令

hugo

3.发现突然出现一个名为public的文件

4.在public的文件夹创建git仓库并将其上传到远程刚刚新建的仓库

5.现在就能在：用户名.github.io的网页上找到自己的博客了
